VAPAR STATIC SITE — HOW TO PUBLISH
==================================
1. Open your repository: github.com/vapar-learn/vapar-learn.github.io
2. DELETE the old index.html (open it -> three dots -> Delete file -> commit).
3. Click Add file -> Upload files.
4. Drag in EVERYTHING inside this folder: all the .html files, styles.css,
   the images folder and the files folder. (Drag the items, not the outer
   folder itself, so index.html sits at the top level.)
5. Commit changes. The site updates at https://vapar-learn.github.io/
   within a couple of minutes.
6. Export your two July 2026 policy briefs to PDF, name them
   policy-brief-1.pdf and policy-brief-2.pdf, and upload them INTO the
   files folder (open the files folder on GitHub first, then Add file).
7. Before cancelling Wix: work through files/pdf-manifest.txt and rescue
   every PDF listed there.
